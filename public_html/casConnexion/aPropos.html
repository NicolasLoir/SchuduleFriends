Le but est simple, créer un site internet et même une application qui permet à un individu de se connecter avec son profil
et quelque truc sur lui. Ces amis ou ces connaissances quel qu’elle soit font la même chose et ils s’ajoutent tous en amis.
Ensuite il suffit à un membre de créer un groupe avec x personnes dedans et chaque personne de ce groupe rentre ces
disponibilités dans le calendrier pour par exemple des vacances, une reunion…

Nous avons crée ce site internet dans le but de réaliser un projet mélangeant html et base de donnée. C'était pour nous une
première expérience dans ces deux domaines informatiques. Ils nous a dans un premier temps permis d'obtenir plusieurs compétences
comme l'html pour créer un site internet mais aussi la partie extérieur du site en utilisant une base de donnée. Dans un second
temps ce projet nous as apporté de nouvelle compétences dans la gestion de projet.
